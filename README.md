# ReactTsWebpackSetting. 
react typescript webpack setting init. 

  
### 순서대로 실행해서 만듬. 
```
npm init
npm i typescript
npm i react react-dom
npm i webpack webpack-cli -D
npm i awesome-typescript-loader -D
npm i @types/react @types/react-dom
```

### build 실행
```
npx webpack
```
